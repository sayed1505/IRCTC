# IRCTC Next - Smart Railway Booking Frontend

This is a responsive, pure HTML/CSS/JS implementation of a modern railway booking interface, inspired by IRCTC. It features a focus on UX improvements, speed, and visual appeal without using external frameworks.

## Features

- **Responsive Navbar**: With hover effects and smooth transitions.
- **Smart Login**: Simple verify check, simulated server delay, and success feedback.
- **Book by Distance**: Dynamic booking windows based on distance slabs with current status (Open/Closed).
- **3D Seat View**: Interactive coach layout with visual seat selection.
- **Smart Booking Wizard**: A step-by-step booking flow with simulated payment processing and refund scenarios.
- **Network Simulation**: A dummy network speed indicator.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Variables, Flexbox, Grid, 3D Transforms, Animations.
- **JavaScript (Vanilla)**: DOM manipulation, State management, Simulation logic.

## How to Run

1. Simply open the `index.html` file in any modern web browser (Chrome, Edge, Firefox).
2. No server or installation required.

## File Structure

- `index.html`: Main structure.
- `style.css`: Styles and animations.
- `script.js`: Application logic.

## Usage Guide

1. **Login**: Click "Login" in the navbar. Enter any details and click Login to see the simulation.
2. **Check Timings**: Hover over "Book by Distance" to see current opening status.
3. **Seat View**: Click "Seat View Booking" to select seats in a 3D environment.
4. **Book a Train**: Click "Smart Book Train" to go through the wizard. Steps will auto-advance on selection or pressing Enter.

Enjoy the next-gen booking experience!
